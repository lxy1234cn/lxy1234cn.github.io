дёӯй—ҙд»¶жҳҜ ASP.NET Core зҡ„ж ёеҝғз»„д»¶пјҢдҫӢеҰӮе“Қеә”зј“еӯҳгҖҒз”ЁжҲ·иә«д»ҪйӘҢиҜҒгҖҒCORSгҖҒSwaggerзӯүйҮҚиҰҒзҡ„еҠҹиғҪйғҪжҳҜз”ұ ASP.NET еҶ…зҪ®зҡ„дёӯй—ҙд»¶жҸҗдҫӣзҡ„

жіЁпјҡиҝҷдёҖзҜҮдё»иҰҒи®Іи§Јдёӯй—ҙд»¶зҡ„дёҖдәӣз®ҖеҚ•жҰӮеҝөпјҢдёӢдёҖзҜҮж–Үз« и®Іи§ЈиҮӘе®ҡд№үдёӯй—ҙд»¶гҖӮ

дёӯй—ҙд»¶зҡ„дёүдёӘжҰӮеҝө MapгҖҒUseгҖҒRun

еӨ„зҗҶйЎәеәҸ

Map з”ЁжқҘе®ҡд№үдёҖдёӘз®ЎйҒ“еҸҜд»ҘеӨ„зҗҶе“ӘдәӣиҜ·жұӮпјҢ Use е’Ң Run з”ЁжқҘе®ҡд№үз®ЎйҒ“пјҢ вҖңдёҖдёӘз®ЎйҒ“вҖқ з”ұ вҖңиӢҘе№ІдёӘUseвҖқ е’Ң вҖңдёҖдёӘRunвҖқ з»„жҲҗпјҢ
жҜҸдёӘ Use еј•е…ҘдёҖдёӘдёӯй—ҙд»¶пјҢиҖҢ Run з”ЁжқҘжү§иЎҢжңҖз»Ҳзҡ„ж ёеҝғеә”з”ЁйҖ»иҫ‘гҖӮ

дёҠд»Јз Ғ

зҢңдёҖдёӢи®ҝй—® /test еҗҺзҡ„иҫ“еҮәеҶ…е®№жҳҜд»Җд№ҲпјҲеҰӮеӣҫпјүпјҹ

иҝҷдёӘйЎәеәҸжҳҜдёҚжҳҜе’ҢеӨ§е®¶жғізҡ„жңүзӮ№дёҚдёҖж ·гҖӮ

дёәд»Җд№ҲдёҚжҳҜиҝҷдёӘйЎәеәҸ

еӣ дёәдёӯй—ҙд»¶жҳҜеҲҶдёәеүҚгҖҒеҗҺйҖ»иҫ‘пјҢиҖҢеүҚеҗҺйҖ»иҫ‘жҳҜз”ұ next еҲҶејҖзҡ„гҖӮ
awit next.Invoke(); жҠҠиҜ·жұӮиҪ¬еҲ°дёӢдёҖдёӘдёӯй—ҙд»¶пјҢеҚідёҚдјҡжү§иЎҢиҜҘиҜӯеҸҘеҗҺйқўзҡ„д»Јз ҒпјҲеҗҺйҖ»иҫ‘пјүпјҢ
зӣҙжҺҘжҠҠиҜ·жұӮиҪ¬еҲ°дёӢдёҖдёӘ вҖңuseвҖқ ;
зӣҙеҲ°йҒҮеҲ° вҖңrunвҖқ , вҖң run вҖң иҙҹиҙЈдёҡеҠЎи§„еҲҷдё”дёҚеҶҚе°ҶиҜ·жұӮеҗ‘еҗҺдј йҖ’пјҢеҪ“ вҖңrunвҖқ жү§иЎҢз»“жқҹеҗҺпјҢ

жіЁпјҡеҰӮжһңжҲ‘д»¬еңЁдёҖдёӘдёӯй—ҙд»¶дёӯдҪҝз”Ё context.Response.WriteAsync зӯүж–№ејҸеҗ‘е®ўжҲ·з«ҜеҸ‘йҖҒе“Қеә”пјҢжҲ‘д»¬е°ұе°ҪйҮҸдёҚеҶҚжү§иЎҢ next.Invoke жҠҠиҜ·жұӮиҪ¬еҲ°е…¶д»–дёӯй—ҙд»¶пјҢ
еӣ дёәе…¶д»–дёӯй—ҙд»¶дёӯжңүеҸҜиғҪеҜ№ Response дёӯиҝӣиЎҢдәҶдҝ®ж”№пјҢжҜ”еҰӮдҝ®ж”№е“Қеә”зҠ¶жҖҒз ҒгҖҒжҠҘж–ҮеӨҙгҖҒеҗ‘жҠҘж–ҮдёӯеҶҷе…Ҙе…¶д»–ж•°жҚ®пјҢиҝҷж ·е°ұдјҡйҖ жҲҗжҠҘж–ҮдҪ“иў«жҚҹеқҸзҡ„й—®йўҳгҖӮ
